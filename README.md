# React-TS-Shopping-App

Shopping App built using React and Typescript. Browse through items, filter results and add items to shoping cart.

## Installation
1. Clone repo 
2. Install NPM packages
    ```
    npm install
    ```
3. Run the program
    ```
    npm run dev
    ```
